# A & A Wedding & Events Planner Website

Production-ready static website for GitHub Pages.

## Upload

1. Extract ZIP.
2. Upload all extracted files directly to GitHub repository root.
3. Settings > Pages > Deploy from branch > main > /root > Save.

## Included

- index.html
- styles.css
- script.js
- assets/images/
